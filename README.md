Core mechanic
Mijn core mechanic is dat de speler naar het volgende blok gaat door een knop (bijv, arrow keys) te klikken.
dit zoorgt voor een sound, en dan spawnt het volgende blok in. dit gaat zo door en door.

Input
De speler doet dit door een van vier knoppen te drukken.

Doel
De speler probeert een zo hoog mogelijke score te krijgen

Easy to learn
De speler begrijpt snel door de icoontjes

Hard to master
Het wordt uitdagend door hoe snel het gaat.
extra fun, dus.

Waarom blijft dit leuk?
Deze actie blijft leuk om te herhalen omdat het elke keer andere knoppen zijn

Juiciness
Ik maak deze actie satisfying door een hogere streak betekent betere muziek/wordt sneller.
